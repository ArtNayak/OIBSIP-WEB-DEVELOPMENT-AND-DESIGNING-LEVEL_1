Personal Portfolio Website

Objective
The objective of this project is to create a personal portfolio website using a Netflix-inspired UI theme to showcase personal details, skills, projects, and contact information.
This project helps beginners understand real-world portfolio structure and styling.

Tools & Technologies Used
HTML5 – Content structure
CSS3 – Styling and layout
Flexbox & CSS Grid – Alignment and project layout
Hover effects & transitions – Interactive UI
No external libraries or frameworks

Steps Performed
Designed a Netflix-style dark theme with red accent colors
Added a hero section with profile image and resume link
Created sections for About, Skills, Projects, and Contact
Implemented project cards in a fixed two-row grid layout
Added clickable email, phone, LinkedIn, and GitHub links

Outcome
A professional-looking personal portfolio website that follows modern UI principles and effectively presents personal and technical details.
The project is suitable for college submissions, interviews, and GitHub hosting.
