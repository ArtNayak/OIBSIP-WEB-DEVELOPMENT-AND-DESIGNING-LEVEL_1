Temperature Converter

Objective
The objective of this project is to build a temperature conversion tool that allows users to convert values between Celsius, Fahrenheit, and Kelvin.
The project introduces basic JavaScript logic along with HTML and CSS.

Tools & Technologies Used
HTML5 – Input fields, buttons, and structure
CSS3 – Google-inspired UI design
JavaScript – Conversion logic and validation

Steps Performed
Created an input field to accept temperature values
Added dropdown menus to select input and output units
Implemented input validation to ensure numeric values
Converted temperatures by normalizing values to Celsius
Displayed the converted temperature with the correct unit

Outcome
A clean, user-friendly temperature converter website with accurate conversions and proper validation.
The project demonstrates practical use of JavaScript functions, conditionals, and DOM manipulation.
